# Gmarciani

Personal blog at [gmarciani.com](https://gmarciani.com).

## Requirements

```
brew install hugo node imagemagick
```

```
npm install
```

Check package updates

```
npx npm-check-updates -u
```

## Usage

Build

```
gulp build
```

Watch

````
gulp watch
````


Start a live preview

```
hugo server -D
```
